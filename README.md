# Flow - template app for gulp and bower

Шаблон для создания приложений, с использованием gulp и bower <br>
В шаблон включены следующие возможности:
  
  - Less препроцессор
  - Autoprefixer
  - Минификация CSS, JS
  - Сжатие изображение
  - Работа с sftp
  
Шаблон включает в себя дополнительные библиотеки

  - Bootstrap 3
  - AngularJS
  - jQuery

Запуск:

```sh
$ gulp watch
```
